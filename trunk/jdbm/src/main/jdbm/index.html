This is the main JDBM package, which contains all the major public classes and interfaces which programmers are expected to use 99% of the time.

Subpackages: jdbm.helper - Helper classes to convert to/from universal Java interfaces jdbm.core - Core JDBM classes used to manage persistent data, pages, etc... jdbm.hash - JDBMHashtable implementation classes jdbm.jta - Contains JTA (Java Transaction API) related wrappers (Planned)
